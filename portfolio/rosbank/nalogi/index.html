Оплата налогов

Web + Mobile, 2023

Сервис оплаты налогов без создания шаблонов, долгого ввода реквизитов

Проблема

ИП и юрлица должны регулярно и вовремя платить налоги. Чтобы это делать, нужно заполнять много реквизитов и делать это без ошибок. Ошибся — налоги не оплачены, деньги зависли. К компании могут быть введены санкции. Решение всех вопросов занимает время, особенно если нет бухгалтера

Контекст

— ФНС вводит единый налоговый счёт

Оплата налогов ИП и юрлиц по одним реквизитам всегда

— У бизнеса появилась идея выводить реквизиты счета

В разделе с платежами показывать реквизи единого налогового счета для их копирования

— Я предложил идею с упрощением пути пользователя

Сделать новый типа платежа, который будет являться готовым шаблоном для оплаты налогов
Быстрая оплата налогов в пару кликов
